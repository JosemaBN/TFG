# Frontend (React + TypeScript + Vite)

Interfaz web del TFG: gestión de **materiales**, **eventos**, **movimientos** y **plantilla** (vista tipo hoja de cálculo).

## Arranque rápido (desarrollo)

### Opción rápida desde la raíz (recomendado)

Desde la **raíz** del repo:

```bash
npm run dev:all
```

1. Instalar dependencias:

```bash
npm install
```

2. Configurar la URL del backend:

- Copia `frontend/.env.example` a `frontend/.env`
- Ajusta `VITE_API_URL` (por defecto `http://127.0.0.1:3000`)

3. Ejecutar Vite:

```bash
npm run dev
```
